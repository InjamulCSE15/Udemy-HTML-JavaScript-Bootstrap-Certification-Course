JavaScript

  • HTML & CSS - Static Programming Languages.
  • JavaScript is Client-Side.
  • Communicates with Web Browser, not Web Server.
  • JavaScript allows User-Web Page Interactivity.
JavaScript does have some limitations:
  • Cannot be forced onto users Web Browser
  • It must be supported on clients web browser
  • Most Browser support JavaScript by default
  • JavaScript is not JAVA
JavaScript is an Object Oriented Language
This means that attributes of an object are contained within variables.
For example:
If we describe a person will refer to this person as the object we would typically use attributes such as: name, height, weight, gender and age.
The values of these attributes will vary or different from person to person. Another word for attributes is variables. These variables will contain the values for each person as we progressed through this discussion.

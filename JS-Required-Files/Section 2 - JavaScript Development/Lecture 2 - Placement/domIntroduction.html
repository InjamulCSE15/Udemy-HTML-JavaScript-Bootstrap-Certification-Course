What is DOM?

  • "Document Object Model"
  • A tree of objects created by your web browser when a web page loads.
  • The web browser uses the DOM to interpret the page and display its contents.


  • Elements can have attributes and text content.
  • Example:
  • anchor tag element
  • Attributes: href, target, id, alt
  • href="http://www.google.com"
  • The DOM allows the elements on a web page to be dynamically changed or updated using JS or jQuery Library.
  • We can change HTML elements, attributes, CSS Styles when different events are triggered.
  • For example: A button is clicked, or a page is loaded.

Three methods of changing elements:
  • By Tag Name (e.g. h1)
  • By ID (e.g. p id="text")
  • By Class Name (e.g. li class="item1")
